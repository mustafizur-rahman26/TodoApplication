# Simple Todo List app

## Provide a basic Model-View-Presenter (MVP) architecture & Repository pattern

## With the app, you can:
- Add new TODOs
- Delete TODOs
- Update TODOs
- Discard changes
- Complete/Incomplete TODOs

## Dependencies
- appcompat-v7:26.1.0
- recyclerview-v7:26.1.0
- cardview-v7:26.1.0
- design:26.1.0
- butterknife-v8.7.0
